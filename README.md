# MedQueue

MedQueue is a full-stack Hospital Queue Management System built using the MERN stack.

## Features

* Role-based Authentication (Admin, Doctor, Patient)
* Appointment Booking System
* Real-Time Queue Management using Socket.IO
* Department Management
* Doctor Management
* Patient Queue Tracking
* Dashboard Analytics
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios
* Socket.IO Client

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Socket.IO
* JWT Authentication
